What I've Built.

⭐ Featured Project

Solar-Wind Hybrid Microgrid with MPPT Control

MATLABPower ElectronicsMPPTSimulink

Designed and simulated a hybrid solar-wind microgrid system with Maximum Power Point Tracking (MPPT) algorithm. Achieved 96.8% energy extraction efficiency. Includes bidirectional DC-DC converter and battery storage management. Presented at National Renewable Energy Conference 2024.

Project 02

Smart Grid Fault Detection System

IoTEmbedded CESP32

IoT-based real-time fault detection for distribution lines using current transformers and ESP32. Sends alerts within 200ms of fault detection. Deployed as a working prototype in college's electrical lab.

Project 03

4-Bit ALU Design using VLSI

VerilogCadenceCMOS

Full 4-bit ALU at transistor level using CMOS in Cadence Virtuoso. Achieved 30% power reduction vs standard design. Includes complete simulation, DRC, and layout verification.
